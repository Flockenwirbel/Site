第三课学习了形容词 duży / mały 及其性数变化规则。本课学习两对高频形容词——**dobry** (好的) 与 **zły** (坏的)、**nowy** (新的) 与 **stary** (旧的)——它们的性数变化完全遵循同一套标准词尾，故以表格为主，辅以少量例句。

---

## 词形总览

### 单数

| 性别 | dobry (好的) | zły (坏的) | nowy (新的) | stary (旧的) | 法语 (bon / mauvais / nouveau / vieux) |
| :--- | :----------- | :--------- | :---------- | :----------- | :------------------------------------- |
| 阳性 | **dobry**    | **zły**    | **nowy**    | **stary**    | bon / mauvais / nouveau / vieux        |
| 阴性 | **dobra**    | **zła**    | **nowa**    | **stara**    | bonne / mauvaise / nouvelle / vieille  |
| 中性 | **dobre**    | **złe**    | **nowe**    | **stare**    | bon / mauvais / nouveau / vieux        |

### 复数

| 类型     | dobry (好的) | zły (坏的) | nowy (新的) | stary (旧的) | 法语                         |
| :------- | :----------- | :--------- | :---------- | :----------- | :--------------------------- |
| 阳性人   | **dobrzy**   | **źli**    | **nowi**    | **starzy**   | bons / mauvais / nouveaux / vieux |
| 非阳性人 | **dobre**    | **złe**    | **nowe**    | **stare**    | bons / mauvais / nouveaux / vieux |

> 四个形容词的词尾完全一致: 阳性 **-y**、阴性 **-a**、中性 **-e**、非阳性人复数 **-e**。仅在阳性人复数中出现辅音交替，见下节。
{: .prompt-info }

---

## 阳性人复数的辅音交替

阳性人复数词尾为 **-i**，导致词干末辅音发生软化：

| 原形   | 阳性人复数 | 交替         | 规律                   |
| :----- | :--------- | :----------- | :--------------------- |
| dobry  | **dobrzy** | r → rz       | r 在 -i 前变为 rz      |
| zły    | **źli**    | z → ź, ł → l | ł 去硬化, z 软化       |
| nowy   | **nowi**   | 无           | 词干无硬化辅音，仅 y→i |
| stary  | **starzy** | r → rz       | 同 dobry               |

> 对比第三课: duży → **duzi** (ż → z)、mały → **mali** (ł → l)。所有交替均为**去硬化**或**软化**现象——硬化辅音在 -i 前还原为对应的普通辅音。
{: .prompt-tip }

---

## 副词

| 波兰语     | 法语         |
| :--------- | :----------- |
| **dobrze** | bien         |
| **źle**    | mal          |
| **nowo**   | nouvellement |
| **staro**  | vieillement  |

> dobrze / źle 为日常高频词。nowo / staro 罕见；"重新" 多用 **od nowa**，"照旧" 多用 **po staremu**。
{: .prompt-tip }

## 例句

### dobry / zły

|        |                           |
| :----- | :------------------------ |
| **PL** | Ten chleb jest **dobry**. |
| **FR** | Ce pain est **bon**.      |

|        |                          |
| :----- | :----------------------- |
| **PL** | Ta zupa jest **dobra**.  |
| **FR** | Cette soupe est bonne.   |

|        |                          |
| :----- | :----------------------- |
| **PL** | To piwo jest **dobre**.  |
| **FR** | Cette bière est bonne.   |

|        |                              |
| :----- | :--------------------------- |
| **PL** | Ta pogoda jest **zła**.      |
| **FR** | Ce temps est mauvais.        |

|        |                                 |
| :----- | :------------------------------ |
| **PL** | To jedzenie jest **złe**.       |
| **FR** | Cette nourriture est mauvaise.  |

|        |                                                          |
| :----- | :------------------------------------------------------- |
| **PL** | Ci lekarze są **dobrzy**, ale tamci kierowcy są **źli**. |
| **FR** | Ces médecins sont bons, mais ces chauffeurs sont mauvais. |

|        |                                            |
| :----- | :----------------------------------------- |
| **PL** | Mówisz **dobrze** po polsku, ale ja **źle**. |
| **FR** | Tu parles bien polonais, mais moi mal.     |

### nowy / stary

|        |                                                 |
| :----- | :---------------------------------------------- |
| **PL** | Ten dom jest **nowy**, a tamten — **stary**.    |
| **FR** | Cette maison est neuve, et celle-là — vieille.  |

|        |                                                  |
| :----- | :----------------------------------------------- |
| **PL** | Ta książka jest **nowa**, ale moja jest **stara**. |
| **FR** | Ton livre est nouveau, mais le mien est vieux.    |

|        |                          |
| :----- | :----------------------- |
| **PL** | To auto jest **nowe**.   |
| **FR** | Cette voiture est neuve. |

|        |                                                               |
| :----- | :------------------------------------------------------------ |
| **PL** | Ci studenci są **nowi**, a tamci profesorowie są **starzy**.  |
| **FR** | Ces étudiants sont nouveaux, et ces professeurs-là sont vieux. |
