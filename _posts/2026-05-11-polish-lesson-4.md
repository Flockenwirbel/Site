本课是形容词专题，系统学习波兰语形容词的性数一致规则，涵盖四对高频形容词: **duży / mały** (大/小)、**dobry / zły** (好/坏)、**nowy / stary** (新/旧).

---

## 形容词性数一致基础

在第二课中我们以 **smaczny** (美味的) 为例初步接触了形容词的性别变化. 波兰语形容词需要与所修饰的名词在**性别**和**数**上保持一致.

### 单数词尾规律

| 性别 | 词尾   | 示例 (smaczny) | 法语         |
| :--- | :----- | :------------- | :----------- |
| 阳性 | **-y** | smaczny sok    | jus délicieux  |
| 阴性 | **-a** | smaczna zupa   | soupe délicieuse |
| 中性 | **-e** | smaczne jabłko | pomme délicieuse |

> 词尾规律: 阳性 **-y** (或 -i, 视词干末辅音而定), 阴性 **-a**, 中性 **-e**. 这与名词的性别标记高度对应.
{: .prompt-info }

### 复数: 阳性人 vs 非阳性人

波兰语形容词复数区分两组: **阳性人** (męskoosobowy, 指男性人类) 和**非阳性人** (niemęskoosobowy, 包括阴性、中性、动物、物体等).

> 这是波兰语语法中最具特色的区分之一. 法语和英语均无此区分, 中文更是完全不存在. 初学者需要特别注意.
{: .prompt-tip }

以下四对形容词将逐一展示单数和复数形式, 并在最后汇总辅音交替规律.

## duży (大的) 与 mały (小的)

### 单数

| 性别 | duży (大的) | mały (小的) | 法语 (grand / petit)  |
| :--- | :---------- | :---------- | :-------------------- |
| 阳性 | **duży**    | **mały**    | grand / petit         |
| 阴性 | **duża**    | **mała**    | grande / petite       |
| 中性 | **duże**    | **małe**    | grand / petit         |

> duży 的词干含 ż (硬化辅音), 因此阳性词尾为 **-y** 而非 -i.
{: .prompt-info }

### 复数

| 类型     | duży      | mały      |
| :------- | :-------- | :-------- |
| 阳性人   | **duzi**  | **mali**  |
| 非阳性人 | **duże**  | **małe**  |

> 阳性人复数发生词干辅音交替: duży → du**z**i (ż → z), mały → ma**l**i (ł → l). 硬化辅音在 -i 前还原为普通辅音.
{: .prompt-tip }

### 例句

**单数**

|        |                          |
| :----- | :----------------------- |
| **PL** | Ten dom jest **duży**.   |
| **FR** | Cette maison est grande. |

|        |                           |
| :----- | :------------------------ |
| **PL** | Tamten kot jest **mały**. |
| **FR** | Ce chat-là est petit.     |

|        |                        |
| :----- | :--------------------- |
| **PL** | Ta ryba jest **mała**. |
| **FR** | Ce poisson est petit.  |

|        |                          |
| :----- | :----------------------- |
| **PL** | To drzewo jest **duże**. |
| **FR** | Cet arbre est grand.     |

**复数**

|        |                             |
| :----- | :-------------------------- |
| **PL** | Ci mężczyźni są **duzi**.   |
| **FR** | Ces hommes sont grands.     |

|        |                            |
| :----- | :------------------------- |
| **PL** | Ci chłopcy są **mali**.    |
| **FR** | Ces garçons sont petits.   |

|        |                           |
| :----- | :------------------------ |
| **PL** | Te psy są **duże**.       |
| **FR** | Ces chiens sont grands.   |

|        |                          |
| :----- | :----------------------- |
| **PL** | Te koty są **małe**.     |

## 阳性人复数的辅音交替

阳性人复数词尾为 **-i**, 导致词干末辅音发生软化或去硬化. 以下是本课六对形容词的汇总:

| 原形    | 阳性人复数  | 交替           | 规律                     |
| :------ | :---------- | :------------- | :----------------------- |
| duży    | **duzi**    | ż → z          | 去硬化                   |
| mały    | **mali**    | ł → l          | 去硬化                   |
| dobry   | **dobrzy**  | r → rz         | r 在 -i 前软化           |
| zły     | **źli**     | z → ź, ł → l   | z 软化 + ł 去硬化        |
| nowy    | **nowi**    | (无)           | 词干无硬化辅音, 仅 y → i |
| stary   | **starzy**  | r → rz         | 同 dobry                 |

> 核心原则: 硬化辅音 (ż, ł 等) 在 -i 前还原为对应的普通辅音 (z, l). 这是波兰语音系中贯穿名词变格和形容词变化的核心规律.
{: .prompt-tip }

## 副词

形容词可派生出副词, 用于修饰动词:

| 形容词   | 副词         | 法语         |
| :------- | :----------- | :----------- |
| dobry    | **dobrze**   | bien         |
| zły      | **źle**      | mal          |
| nowy     | **nowo**     | nouvellement |
| stary    | **staro**    | vieillement  |
| duży     | **dużo**     | beaucoup     |
| mały     | **mało**     | peu          |

> dobrze / źle 为日常高频词. nowo / staro 罕见; "重新" 多用 **od nowa**, "照旧" 多用 **po staremu**. dużo / mało 常用于数量表达, 如 **dużo wody** (很多水), **mało czasu** (很少时间).
{: .prompt-tip }

### 副词例句

|        |                                                       |
| :----- | :---------------------------------------------------- |
| **PL** | Mówisz **dobrze** po polsku, ale ja mówię **źle**.    |
| **FR** | Tu parles **bien** polonais, mais je parle **mal**.   |

|        |                                                   |
| :----- | :------------------------------------------------ |
| **PL** | On ma **dużo** pieniędzy, a ja mam **mało**.      |
| **FR** | Il a **beaucoup** d'argent, et j'en ai **peu**.   |

## 综合练习

### 场景一: 在餐厅

|        |                                                                                   |
| :----- | :-------------------------------------------------------------------------------- |
| **PL** | Ta zupa jest **dobra**, ale tamta kawa jest **zła**. Ten chleb jest **świeży**.   |
| **FR** | Cette soupe est bonne, mais ce café-là est mauvais. Ce pain est frais.            |

> 练习要点: 形容词性数配合 (dobra 阴性, zła 阴性, świeży 阳性), 近指远指对比 (ta / tamta).
{: .prompt-tip }

### 场景二: 比较两栋房子

|        |                                                                                         |
| :----- | :-------------------------------------------------------------------------------------- |
| **PL** | Ten dom jest **duży** i **nowy**. Tamten dom jest **mały** i **stary**.                  |
| **FR** | Cette maison est grande et neuve. Cette maison-là est petite et vieille.                |

> 练习要点: 多形容词连用 (duży + nowy 同为阳性, 修饰 dom), 近指远指对比 (ten / tamten).
{: .prompt-tip }

### 场景三: 描述一群人

|        |                                                                                    |
| :----- | :--------------------------------------------------------------------------------- |
| **PL** | Ci studenci są **mali**, ale **dobrzy**. Tamci profesorowie są **duzi** i **starzy**. |
| **FR** | Ces étudiants sont petits, mais bons. Ces professeurs-là sont grands et vieux.      |

> 练习要点: 阳性人复数形容词 (mali、dobrzy、duzi、starzy), 辅音交替 (mały → mali, dobry → dobrzy, duży → duzi, stary → starzy).
{: .prompt-tip }

## 语法总结

### 形容词变化步骤

**步骤1: 确定名词性别 (单数)**

| 性别 | 词尾 | duży  | mały  | dobry | zły  | nowy  | stary |
| :--- | :--- | :---- | :---- | :---- | :--- | :---- | :---- |
| 阳性 | -y   | duży  | mały  | dobry | zły  | nowy  | stary |
| 阴性 | -a   | duża  | mała  | dobra | zła  | nowa  | stara |
| 中性 | -e   | duże  | małe  | dobre | złe  | nowe  | stare |

**步骤2: 确定复数类型**

- 阳性人名词 → 阳性人复数 (词尾 **-i**, 可能引发辅音交替)
- 其余名词 → 非阳性人复数 (词尾 **-e**, 与中性单数同形)

| 类型     | duży  | mały  | dobry   | zły  | nowy  | stary   |
| :------- | :---- | :---- | :------ | :--- | :---- | :------ |
| 阳性人   | duzi  | mali  | dobrzy  | źli  | nowi  | starzy  |
| 非阳性人 | duże  | małe  | dobre   | złe  | nowe  | stare   |

> 初学建议: 先掌握单数三性 (阳性 -y, 阴性 -a, 中性 -e), 再逐步引入复数两组. 辅音交替在遇到具体词汇时逐个记忆即可, 不必一次性穷举所有规律.
{: .prompt-info }
