## 五个常用动词与变位

以下是第二课需要掌握的五个常用动词. 波兰语动词变位有较强的规律性, 学会这五个, 日常基本对话就有了基础.

### jeść — 吃

| 人称    | 变位      |
| :------ | :-------- |
| ja      | **jem**   |
| ty      | **jesz**  |
| on/ona  | **je**    |
| my      | **jemy**  |
| wy      | **jecie** |
| oni/one | **jedzą** |

### pić — 喝

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **piję**    |
| ty      | **pijesz**  |
| on/ona  | **pije**    |
| my      | **pijemy**  |
| wy      | **pijecie** |
| oni/one | **piją**    |

### mieć — 有

| 人称    | 变位      |
| :------ | :-------- |
| ja      | **mam**   |
| ty      | **masz**  |
| on/ona  | **ma**    |
| my      | **mamy**  |
| wy      | **macie** |
| oni/one | **mają**  |

### lubić — 喜欢

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **lubię**   |
| ty      | **lubisz**  |
| on/ona  | **lubi**    |
| my      | **lubimy**  |
| wy      | **lubicie** |
| oni/one | **lubią**   |

### mówić — 说

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **mówię**   |
| ty      | **mówisz**  |
| on/ona  | **mowi**    |
| my      | **mówimy**  |
| wy      | **mówicie** |
| oni/one | **mówią**   |

用 mówić 表达"以某种语言说话"时, 使用 **po + 副词** 的结构:

| 波兰语           | 法语        |
| :--------------- | :---------- |
| **po polsku**    | en polonais |
| **po angielsku** | en anglais  |
| **po niemiecku** | en allemand |
| **po francusku** | en français |
| **po chińsku**   | en chinois  |
| **po rosyjsku**  | en russe    |
| **po włosku**    | en italien  |

例句:

|        |                      |
| :----- | :------------------- |
| **PL** | Ona mówi po polsku.  |
| **FR** | Elle parle polonais. |

> 注意观察变位规律: **ty (你)** 的变位通常以 **-sz** 结尾 (jesz, pijesz, masz, lubisz, mówisz), **my (我们)** 以 **-my** 结尾, **wy (你们)** 以 **-cie** 结尾. 这套规律适用于绝大多数波兰语动词, 相比德语的强/弱变化要规则得多.

## 动物词汇

波兰语中"动物"一词是 **zwierzę** (中性名词). 以下是常见的动物词汇:

| 波兰语     | 法语       |
| :--------- | :--------- |
| **pies**   | le chien   |
| **kot**    | le chat    |
| **koń**    | le cheval  |
| **słoń**   | l'éléphant |
| **kaczka** | le canard  |
| **lew**    | le lion    |
| **mysz**   | la souris  |
| **tygrys** | le tigre   |
| **pająk**  | l'araignée |
| **ptak**   | l'oiseau   |
| **ryba**   | le poisson |

> 注意动物名词的性别: pies、kot、koń、słoń、lew、tygrys、pająk、ptak、ryba 以辅音结尾, 为阳性；kaczka 以 -a 结尾, 为阴性. **mysz 虽然以辅音结尾, 但也为阴性**. 动物名词的性别不一定与实际生物性别对应.
{: .prompt-tip }

## 形容词的性数一致

波兰语的形容词需要与所修饰的名词在性别和数上保持一致. 以 **smaczny** (délicieux) 为例:

| 搭配               | 性别 |
| :----------------- | :--- |
| **smaczny sok**    | 阳性 |
| **smaczne wino**   | 中性 |
| **smaczna zupa**   | 阴性 |
| **smaczne jabłko** | 中性 |

> 形容词词尾变化规律: 阳性为 **-y** (或 -i), 阴性为 **-a**, 中性为 **-e** (或 -ie). 这与名词的性别标记高度对应.
{: .prompt-info }

## 名词复数: 词尾的变化

法语的复数通常只需加 -s, 波兰语的复数变化则更为多样. 以下是几组常见例子:

| 单数            | 复数            | 规律提示       |
| :-------------- | :-------------- | :------------- |
| **kobieta**     | **kobiety**     | 阴性 -a → -y   |
| **mężczyzna**   | **mężczyźni**   | 阳性, 内部变音 |
| **chłopiec**    | **chłopcy**     | 阳性, 词尾变化 |
| **dziewczynka** | **dziewczynki** | 阴性 -a → -i   |
| **dziecko**     | **dzieci**      | 中性, 不规则   |
| **kot**         | **koty**        | 阳性, 加 -y    |
| **pies**        | **psy**         | 阳性, 不规则   |

## 句型示例

### 动作句

|        |                     |
| :----- | :------------------ |
| **PL** | On je **jabłko**.   |
| **FR** | Il mange une pomme. |

|        |                            |
| :----- | :------------------------- |
| **PL** | **Chłopiec** je **chleb**. |
| **FR** | Le garçon mange du pain.   |

### 定义句 — "……是……"

用 **to** 连接主语和定义时, 名词保持主格形式:

|        |                         |
| :----- | :---------------------- |
| **PL** | Warszawa to **miasto**. |
| **FR** | Varsovie est une ville. |

> 注意 **to** 的用法: **Warszawa to miasto**. 这里 to 起系动词作用, 相当于"is". 这是波兰语中非常常见且实用的句式.
{: .prompt-tip }

### 工具格与 "to" 连用

用 **To jest** 定义身份时, 若后面接工具格形式:

|        |                                             |
| :----- | :------------------------------------------ |
| **PL** | To jest **tygrys**. → To jest **tygrysem**. |
| **FR** | C'est un tigre.                             |

> 当使用 "to jest + 工具格" 时, 表示"这是 (一个) ……". 工具格词尾变化因性别而异: 阳性辅音结尾通常加 **-em** (tygrys → tygrysem), 阴性 -a 变为 **-ą** (kobieta → kobietą), 中性 -o 变为 **-em** (dziecko → dzieckiem). 这个规则将在后续课程中详细展开.
{: .prompt-info }

## 综合练习: 餐桌场景

现在把本课学到的知识放进生活场景里.

### 场景一: 早餐时间

|        |                                                                             |
| :----- | :-------------------------------------------------------------------------- |
| **PL** | Na śniadanie jem **smaczny chleb** i piję **herbatę**. Ona pije **sok**.    |
| **FR** | Au petit-déjeuner je mange du bon pain et je bois du thé. Elle boit du jus. |

> 练习要点: 三餐 (śniadanie) 、食品词汇 (chleb) 、饮品词汇 (herbata, sok) 、形容词一致 (smaczny + chleb 阳性) 、jeść / pić 的第一和第三人称变位 (jem / piję / pije) 、阴性宾格 (herbata → herbatę).
{: .prompt-tip }

### 场景二: 在餐厅

|        |                                                                        |
| :----- | :--------------------------------------------------------------------- |
| **PL** | Ja lubię **kawę**, a on lubi **herbatę**. My mamy **chleb** i **ser**. |
| **FR** | J'aime le café, et il aime le thé. Nous avons du pain et du fromage.   |

> 练习要点: lubić 变位 (lubię / lubi) 、mieć 变位 (mamy) 、食品词汇 (chleb, ser) 、饮品词汇 (kawa, herbata) 、阴性宾格 (kawa → kawę, herbata → herbatę).
{: .prompt-tip }
