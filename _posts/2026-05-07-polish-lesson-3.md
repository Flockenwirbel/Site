在前两课中我们学习了名词系统、动词变位和基础句型. 第三课将深入指示代词系统, 学习如何根据距离和性别选择正确的形式, 同时学习两个常用形容词 duży (大的) 与 mały (小的) 的性数变化.

---

## 指示代词: to 与 tamto

波兰语的指示代词区分距离: **to** (这/这个) 和 **tamto** (那/那个).

### 近指与远指对照

| 波兰语    | 法语             |
| :-------- | :--------------- |
| **to**    | ce / ça          |
| **tamto** | ça-là / celui-là |

### 用法示例

**近指 (to) — 这是……**

|        |                       |
| :----- | :-------------------- |
| **PL** | **To** jest dom.      |
| **FR** | **C'est** une maison. |

**远指 (tamto) — 那是……**

|        |                          |
| :----- | :----------------------- |
| **PL** | **Tamto** jest dom.      |
| **FR** | **Cela** est une maison. |

**区分近指与远指**

|        |                                                 |
| :----- | :---------------------------------------------- |
| **PL** | **To** jest kot. **Tamto** jest pies.           |
| **FR** | **C'est** un chat. **C'est** un chien (là-bas). |

## 指示代词的性数配合

当指示代词直接修饰名词时, 需要根据名词的**性别**和**距离**来选择正确的形式.

### 近指: 这/这个

| 性别 | 波兰语  | 法语     |
| :--- | :------ | :------- |
| 阳性 | **ten** | ce / cet |
| 阴性 | **ta**  | cette    |
| 中性 | **to**  | ce / cet |

> **注意**: 这里的 **to** 是指示代词, 需要与名词的性别配合. 而 **To jest...** 句型中的 to 是不变格的形式, 意为"这是", 两者用法不同.
{: .prompt-tip }

### 远指: 那/那个

| 性别 | 波兰语     | 法语     |
| :--- | :--------- | :------- |
| 阳性 | **tamten** | celui-là |
| 阴性 | **tamta**  | celle-là |
| 中性 | **tamto**  | cela     |

### 用法示例

**阳性名词**

|        |                         |
| :----- | :---------------------- |
| **PL** | **Ten** pies jest duży. |
| **FR** | **Ce** chien est grand. |

|        |                            |
| :----- | :------------------------- |
| **PL** | **Tamten** pies jest mały. |
| **FR** | **Ce** chien-là est petit. |

**阴性名词**

|        |                                 |
| :----- | :------------------------------ |
| **PL** | **Ta** kobieta mówi po polsku.  |
| **FR** | **Cette** femme parle polonais. |

|        |                                      |
| :----- | :----------------------------------- |
| **PL** | **Tamta** kobieta mówi po angielsku. |
| **FR** | **Cette** femme-là parle anglais.    |

**中性名词**

|        |                                 |
| :----- | :------------------------------ |
| **PL** | **To** dziecko je jabłko.       |
| **FR** | **Cet** enfant mange une pomme. |

|        |                                  |
| :----- | :------------------------------- |
| **PL** | **Tamto** dziecko pije wodę.     |
| **FR** | **Cet** enfant-là boit de l'eau. |

### 对比练习: 近指 vs 远指

|        |                                                                   |
| :----- | :---------------------------------------------------------------- |
| **PL** | **Ten** dom jest duży, ale **tamten** dom jest mały.              |
| **FR** | **Cette** maison est grande, mais **cette** maison-là est petite. |

> 当句子中同时出现近指和远指时, 使用对比结构: **ten... ale tamten...** (这个...但那个...). 这种结构在描述两个事物时非常实用.
{: .prompt-tip }

## 指示代词的复数形式

指示代词也需要与名词的数保持一致. 波兰语的复数系统区分**阳性人** (阳性人类名词) 和**其余** (阴性、中性、非人类阳性).

### 近指复数: 这些

| 类型   | 波兰语 | 法语 |
| :----- | :----- | :--- |
| 阳性人 | **ci** | ces  |
| 其余   | **te** | ces  |

### 远指复数: 那些

| 类型   | 波兰语    | 法语          |
| :----- | :-------- | :------------ |
| 阳性人 | **tamci** | ceux-là       |
| 其余   | **tamte** | ceux-là / ces |

### 复数用法示例

**阳性人名词** (mężczyźni - 男人们)

|        |                                   |
| :----- | :-------------------------------- |
| **PL** | **Ci** mężczyźni mówią po polsku. |
| **FR** | **Ces** hommes parlent polonais.  |

**其余复数名词** (kobiety - 女人们, psy - 狗, dzieci - 儿童)

|        |                                |
| :----- | :----------------------------- |
| **PL** | **Te** kobiety lubią kawę.     |
| **FR** | **Ces** femmes aiment le café. |

|        |                             |
| :----- | :-------------------------- |
| **PL** | **Te** psy są duże.         |
| **FR** | **Ces** chiens sont grands. |

|        |                                      |
| :----- | :----------------------------------- |
| **PL** | **Te** dzieci jedzą obiad.           |
| **FR** | **Ces** enfants mangent le déjeuner. |

**远指复数对比**

|        |                                                                     |
| :----- | :------------------------------------------------------------------ |
| **PL** | **Ci** studenci studiują polonistykę, a **tamci** — anglistykę.     |
| **FR** | **Ces** étudiants étudient le polonais, et **ceux-là** — l'anglais. |

## 形容词: duży 与 mały

在第二课中我们学习了形容词的基本性数规则: 阳性 **-y**, 阴性 **-a**, 中性 **-e**. 本课将用两个高频形容词 **duży** (大的) 和 **mały** (小的) 来巩固这一规则, 并扩展到复数形式.

### 单数形式

| 性别 | duży (大的) | 法语   | mały (小的) | 法语   |
| :--- | :---------- | :----- | :---------- | :----- |
| 阳性 | **duży**    | grand  | **mały**    | petit  |
| 阴性 | **duża**    | grande | **mała**    | petite |
| 中性 | **duże**    | grand  | **małe**    | petit  |

> duży 和 mały 都严格遵循第二课学到的词尾规律: 阳性 **-y**, 阴性 **-a**, 中性 **-e**. 注意 duży 的阳性形式因为词干包含 ż (硬化辅音), 所以词尾是 **-y** 而不是 -i.
{: .prompt-info }

### 复数形式

波兰语形容词复数区分**阳性人** (mężkoosobowy) 和**非阳性人** (niemężkoosobowy) 两组:

| 类型     | duży (大的) | mały (小的) |
| :------- | :---------- | :---------- |
| 阳性人   | **duzi**    | **mali**    |
| 非阳性人 | **duże**    | **małe**    |

> 阳性人复数发生了**词干元音交替**: duży → du**z**i (ż → z), mały → ma**l**i. 这是波兰语中常见的现象, 当词尾变为 -i 时, 硬化辅音要还原为对应的普通辅音.
{: .prompt-tip }

### 用法示例

**阳性单数**

|        |                          |
| :----- | :----------------------- |
| **PL** | Ten dom jest **duży**.   |
| **FR** | Cette maison est grande. |

|        |                           |
| :----- | :------------------------ |
| **PL** | Tamten kot jest **mały**. |
| **FR** | Ce chat-là est petit.     |

**阴性单数**

|        |                                   |
| :----- | :-------------------------------- |
| **PL** | Ta kobieta ma **dużą** rodzinę.   |
| **FR** | Cette femme a une grande famille. |

|        |                        |
| :----- | :--------------------- |
| **PL** | Ta ryba jest **mała**. |
| **FR** | Ce poisson est petit.  |

> 注意: 形容词作定语时也需要配合性别. **duża** 修饰阴性名词 rodzinę (宾格形式, 主格 rodzina), 即使名词变格了, 形容词的阴性词尾 -a 依然保留.
{: .prompt-tip }

**中性单数**

|        |                           |
| :----- | :------------------------ |
| **PL** | To dziecko jest **małe**. |
| **FR** | Cet enfant est petit.     |

|        |                          |
| :----- | :----------------------- |
| **PL** | To drzewo jest **duże**. |
| **FR** | Cet arbre est grand.     |

**阳性人复数**

|        |                           |
| :----- | :------------------------ |
| **PL** | Ci mężczyźni są **duzi**. |
| **FR** | Ces hommes sont grands.   |

|        |                          |
| :----- | :----------------------- |
| **PL** | Ci chłopcy są **mali**.  |
| **FR** | Ces garçons sont petits. |

**非阳性人复数**

|        |                         |
| :----- | :---------------------- |
| **PL** | Te psy są **duże**.     |
| **FR** | Ces chiens sont grands. |

|        |                        |
| :----- | :--------------------- |
| **PL** | Te koty są **małe**.   |
| **FR** | Ces chats sont petits. |

### 对比练习: duży vs mały

|        |                                                       |
| :----- | :---------------------------------------------------- |
| **PL** | Ten słoń jest **duży**, ale ta mysz jest **mała**.    |
| **FR** | Cet éléphant est grand, mais cette souris est petite. |

|        |                                                            |
| :----- | :--------------------------------------------------------- |
| **PL** | Ci studenci są **duzi**, a te studentki są **małe**.       |
| **FR** | Ces étudiants sont grands, et ces étudiantes sont petites. |

## 综合练习: 场景应用

### 场景一: 在公园

|        |                                                                              |
| :----- | :--------------------------------------------------------------------------- |
| **PL** | **To** jest **pies**. **Te** psy są duże. **Tamto** jest **drzewo**.         |
| **FR** | **C'est** un chien. **Ces** chiens sont grands. **C'est** un arbre (là-bas). |

> 练习要点: 不变格指示代词 (To jest, Tamto jest) 、性数配合指示代词 (Te psy 阳性复数) 、动物词汇 (pies) 、形容词一致 (duże).
{: .prompt-tip }

### 场景二: 在咖啡馆

|        |                                                                                    |
| :----- | :--------------------------------------------------------------------------------- |
| **PL** | **Ta** kawa jest dobra. **Tamta** kawa jest lepsza. **To** ciasteczko jest pyszne. |
| **FR** | **Ce** café est bon. **Ce** café-là est meilleur. **Ce** biscuit est délicieux.    |

> 练习要点: 近指/远指对比 (ta / tamta), 性别配合 (kawa 阴性 → ta, ciasteczko 中性 → to), 形容词位置.
{: .prompt-tip }

### 场景三: 描述一家人

|        |                                                                                              |
| :----- | :------------------------------------------------------------------------------------------- |
| **PL** | **Ten** mężczyzna to mój ojciec. **Ta** kobieta to moja matka. **Ci** chłopcy to moi bracia. |
| **FR** | **Cet** homme est mon père. **Cette** femme est ma mère. **Ces** garçons sont mes frères.    |

> 练习要点: 阳性名词配合 (mężczyzna → ten, chłopcy → ci), 阴性名词配合 (kobieta → ta), 家庭词汇.
{: .prompt-tip }

## 语法总结

### 指示代词选择规律

**步骤1: 确定距离**
- 近指 (这个): ten / ta / to (单数), ci / te (复数)
- 远指 (那个): tamten / tamta / tamto (单数), tamci / tamte (复数)

**步骤2: 确定名词性别 (单数)**
- 阳性名词 → ten / tamten
- 阴性名词 → ta / tamta
- 中性名词 → to / tamto

**步骤3: 确定复数类型 (复数)**
- 阳性人名词 (男人、男孩、学生等) → ci / tamci
- 其余名词 (阴性、中性、动物等) → te / tamte

> 初学者最容易犯的错误是在需要配合性数时使用不变格的 to. 记住: **To jest dom** (正确) vs **To dom jest duży** (错误, 应为 **Ten dom jest duży**).
{: .prompt-warning }

### 形容词性数配合规律

**步骤1: 确定名词性别 (单数)**

以 duży / mały 为例:

| 性别 | 词尾 | duży | mały |
| :--- | :--- | :--- | :--- |
| 阳性 | -y   | duży | mały |
| 阴性 | -a   | duża | mała |
| 中性 | -e   | duże | małe |

**步骤2: 确定复数类型 (复数)**

| 类型     | duży | mały |
| :------- | :--- | :--- |
| 阳性人   | duzi | mali |
| 非阳性人 | duże | małe |

> 阳性人复数的形容词词尾为 **-i**, 此时词干可能发生辅音交替: ż → z (duży → duzi), ł → l (mały → mali). 非阳性人复数与中性单数形式相同, 均为 **-e**.
{: .prompt-info }
