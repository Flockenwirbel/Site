在前两课中我们学习了名词系统、动词变位和基础句型. 第三课将深入指示代词系统, 学习如何根据距离和性别选择正确的形式.

---

## 指示代词: to 与 tamto

波兰语的指示代词区分距离: **to** (这/这个) 和 **tamto** (那/那个).

### 近指与远指对照

| 波兰语    | 法语             |
| :-------- | :--------------- |
| **to**    | ce / ça          |
| **tamto** | ça-là / celui-là |

### 用法示例

**近指 (to) — 这是……**

|        |                       |
| :----- | :-------------------- |
| **PL** | **To** jest dom.      |
| **FR** | **C'est** une maison. |

**远指 (tamto) — 那是……**

|        |                          |
| :----- | :----------------------- |
| **PL** | **Tamto** jest dom.      |
| **FR** | **Cela** est une maison. |

**区分近指与远指**

|        |                                                 |
| :----- | :---------------------------------------------- |
| **PL** | **To** jest kot. **Tamto** jest pies.           |
| **FR** | **C'est** un chat. **C'est** un chien (là-bas). |

## 指示代词的性数配合

当指示代词直接修饰名词时, 需要根据名词的**性别**和**距离**来选择正确的形式.

### 近指: 这/这个

| 性别 | 波兰语  | 法语     |
| :--- | :------ | :------- |
| 阳性 | **ten** | ce / cet |
| 阴性 | **ta**  | cette    |
| 中性 | **to**  | ce / cet |

> **注意**: 这里的 **to** 是指示代词, 需要与名词的性别配合. 而 **To jest...** 句型中的 to 是不变格的形式, 意为"这是", 两者用法不同.
{: .prompt-tip }

### 远指: 那/那个

| 性别 | 波兰语     | 法语     |
| :--- | :--------- | :------- |
| 阳性 | **tamten** | celui-là |
| 阴性 | **tamta**  | celle-là |
| 中性 | **tamto**  | cela     |

### 用法示例

**阳性名词**

|        |                               |
| :----- | :---------------------------- |
| **PL** | **Ten** pies je mięso.        |
| **FR** | **Ce** chien mange de la viande. |

|        |                                    |
| :----- | :--------------------------------- |
| **PL** | **Tamten** kot pije mleko.         |
| **FR** | **Ce** chat-là boit du lait.       |

**阴性名词**

|        |                                 |
| :----- | :------------------------------ |
| **PL** | **Ta** kobieta mówi po polsku.  |
| **FR** | **Cette** femme parle polonais. |

|        |                                      |
| :----- | :----------------------------------- |
| **PL** | **Tamta** kobieta mówi po angielsku. |
| **FR** | **Cette** femme-là parle anglais.    |

**中性名词**

|        |                                 |
| :----- | :------------------------------ |
| **PL** | **To** dziecko je jabłko.       |
| **FR** | **Cet** enfant mange une pomme. |

|        |                                  |
| :----- | :------------------------------- |
| **PL** | **Tamto** dziecko pije wodę.     |
| **FR** | **Cet** enfant-là boit de l'eau. |

### 对比练习: 近指 vs 远指

|        |                                                                         |
| :----- | :---------------------------------------------------------------------- |
| **PL** | **Ten** dom jest tutaj, ale **tamten** dom jest tam.                    |
| **FR** | **Cette** maison est ici, mais **cette** maison-là est là-bas.          |

> 当句子中同时出现近指和远指时, 使用对比结构: **ten... ale tamten...** (这个...但那个...). 这种结构在描述两个事物时非常实用.
{: .prompt-tip }

## 指示代词的复数形式

指示代词也需要与名词的数保持一致. 波兰语的复数系统区分**阳性人** (阳性人类名词) 和**其余** (阴性、中性、非人类阳性).

### 近指复数: 这些

| 类型   | 波兰语 | 法语 |
| :----- | :----- | :--- |
| 阳性人 | **ci** | ces  |
| 其余   | **te** | ces  |

### 远指复数: 那些

| 类型   | 波兰语    | 法语          |
| :----- | :-------- | :------------ |
| 阳性人 | **tamci** | ceux-là       |
| 其余   | **tamte** | ceux-là / ces |

### 复数用法示例

**阳性人名词** (mężczyźni - 男人们)

|        |                                   |
| :----- | :-------------------------------- |
| **PL** | **Ci** mężczyźni mówią po polsku. |
| **FR** | **Ces** hommes parlent polonais.  |

**其余复数名词** (kobiety - 女人们, psy - 狗, dzieci - 儿童)

|        |                                |
| :----- | :----------------------------- |
| **PL** | **Te** kobiety lubią kawę.     |
| **FR** | **Ces** femmes aiment le café. |

|        |                             |
| :----- | :-------------------------- |
| **PL** | **Te** psy jedzą mięso.     |
| **FR** | **Ces** chiens mangent de la viande. |

|        |                                      |
| :----- | :----------------------------------- |
| **PL** | **Te** dzieci jedzą obiad.           |
| **FR** | **Ces** enfants mangent le déjeuner. |

**远指复数对比**

|        |                                                                     |
| :----- | :------------------------------------------------------------------ |
| **PL** | **Ci** studenci studiują polonistykę, a **tamci** — anglistykę.     |
| **FR** | **Ces** étudiants étudient le polonais, et **ceux-là** — l'anglais. |

## 综合练习: 场景应用

### 场景一: 在公园

|        |                                                                              |
| :----- | :--------------------------------------------------------------------------- |
| **PL** | **To** jest **pies**. **Tamto** jest **kot**. **Te** psy są tutaj.           |
| **FR** | **C'est** un chien. **C'est** un chat (là-bas). **Ces** chiens sont ici.     |

> 练习要点: 不变格指示代词 (To jest, Tamto jest) 、性数配合指示代词 (Te psy 阳性复数) 、动物词汇 (pies, kot).
{: .prompt-tip }

### 场景二: 在咖啡馆

|        |                                                                                    |
| :----- | :--------------------------------------------------------------------------------- |
| **PL** | **Ta** kawa to espresso. **Tamta** herbata to zielona. **To** jest woda.            |
| **FR** | **Ce** café est un espresso. **Ce** thé-là est vert. **C'est** de l'eau.            |

> 练习要点: 近指/远指对比 (ta / tamta), 性别配合 (kawa 阴性 → ta, herbata 阴性 → tamta), to jest 句型.
{: .prompt-tip }

### 场景三: 描述一家人

|        |                                                                                              |
| :----- | :------------------------------------------------------------------------------------------- |
| **PL** | **Ten** mężczyzna to mój ojciec. **Ta** kobieta to moja matka. **Ci** chłopcy to moi bracia. |
| **FR** | **Cet** homme est mon père. **Cette** femme est ma mère. **Ces** garçons sont mes frères.    |

> 练习要点: 阳性名词配合 (mężczyzna → ten, chłopcy → ci), 阴性名词配合 (kobieta → ta), 家庭词汇.
{: .prompt-tip }

## 语法总结

### 指示代词选择规律

**步骤1: 确定距离**
- 近指 (这个): ten / ta / to (单数), ci / te (复数)
- 远指 (那个): tamten / tamta / tamto (单数), tamci / tamte (复数)

**步骤2: 确定名词性别 (单数)**
- 阳性名词 → ten / tamten
- 阴性名词 → ta / tamta
- 中性名词 → to / tamto

**步骤3: 确定复数类型 (复数)**
- 阳性人名词 (男人、男孩、学生等) → ci / tamci
- 其余名词 (阴性、中性、动物等) → te / tamte

> 初学者最容易犯的错误是在需要配合性数时使用不变格的 to. 记住: **To jest dom** (正确, 意为"这是一栋房子") vs **To dom** (缺少动词, 不完整). 当 to 作为指示形容词修饰名词时, 必须配合性别: **Ten dom** jest tutaj.
{: .prompt-warning }
