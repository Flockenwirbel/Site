学波兰语的起点是肖邦. 但真正让我留下来的,  是波兰的历史本身. 一个在宗教与政治上曾经出奇宽容的国家,  在十七世纪以翼骑兵驰骋欧洲、解围维也纳,  然后在此后的两百年里几度从地图上消失,  却从未被遗忘. 

---

## 认识波兰语的"性别世界"

波兰语的词性分为三种: 阳性 (męski) 、阴性 (żeński) 和中性 (nijaki). 

先来看一组基础词汇. 

| 波兰语          | 德语        | 法语       | 含义        |
| :-------------- | :---------- | :--------- | :---------- |
| **mężczyzna**   | der Mann    | un homme   | 男人 (阳性) |
| **chłopiec**    | der Junge   | un garçon  | 男孩 (阳性) |
| **kobieta**     | die Frau    | une femme  | 女人 (阴性) |
| **dziewczynka** | das Mädchen | une fille  | 女孩 (阴性) |
| **dziecko**     | das Kind    | un enfant  | 儿童 (中性) |
| **człowiek**    | der Mensch  | un humain  | 人类 (阳性) |
| **mleko**       | die Milch   | du lait    | 牛奶 (中性) |
| **chleb**       | das Brot    | du pain    | 面包 (阳性) |
| **woda**        | das Wasser  | de l'eau   | 水 (阴性)   |
| **ciasteczko**  | der Keks    | un biscuit | 饼干 (中性) |

> 以  **-a** 结尾的名词通常是阴性,  以 **-o** 或 **-e** 结尾的通常是中性,  以辅音结尾的则多为阳性. 不过例外也不少——mężczyzna (男人) 以 -a 结尾,  却是阳性名词. 
{: .prompt-tip }

## 格变化: 名词的形态变化

波兰语和德语有一个共同特点: 名词在句子中承担不同语法角色时,  词尾会随之变化. 法语则不同——名词本身不变形,  只有代词会变. 

阴性名词 (-a 结尾) 作宾语时,  词尾 -a 变为 -ę. 

|        |                               |
| :----- | :---------------------------- |
| **PL** | On pije **wodę**.             |
| **DE** | Er trinkt **das Wasser**.     |
| **FR** | Il boit **de l'eau**.         |
| **中** | 他喝水.  (woda 作宾语 → wodę) |

而中性名词作宾语时,  形式保持不变: 

|        |                                         |
| :----- | :-------------------------------------- |
| **PL** | On je **jabłko**.                       |
| **DE** | Er isst **den Apfel**.                  |
| **FR** | Il mange **une pomme**.                 |
| **中** | 他吃苹果.  (jabłko 中性,  宾格形式不变) |

> 波兰语的格变化体系比德语更为复杂——德语有四格,  波兰语有七格. 不过第一课只需要认识**主格 (Mianownik)**、**宾格 (Biernik)**、**工具格 (Narzędnik)** 即可. 
{: .prompt-info }

## "我是……" — być 动词变位 `être / sein`

在大多数语言中,  "be 动词"都是最常用、也最需要优先掌握的不规则动词. 波兰语的 **być** 对应法语 être 和德语 sein. 

六个人称的变位形式——三语对照如下: 

| 人称    | 波兰语 być    | 德语 sein | 法语 être |
| :------ | :------------ | :-------- | :-------- |
| ja      | **jestem**    | bin       | suis      |
| ty      | **jesteś**    | bist      | es        |
| on,ona  | **jest**      | ist       | est       |
| my      | **jesteśmy**  | sind      | sommes    |
| wy      | **jesteście** | seid      | êtes      |
| oni,one | **są**        | sind      | sont      |

> 用 być 表达身份时,  后面的名词需要使用**工具格 (Narzędnik)**. 例如 Ja jestem kobietą (我是女人) ——kobietą 是 kobieta 的工具格形式. 这一点与德语和法语不同,  后两者在系动词后直接使用主格或表语名词. 
{: .prompt-tip }

## 五个常用动词与变位

以下是第一课需要掌握的五个常用动词. 波兰语动词变位有较强的规律性,  学会这五个,  日常基本对话就有了基础. 

### jeść — 吃 `manger / essen`

| 人称    | 变位      |
| :------ | :-------- |
| ja      | **jem**   |
| ty      | **jesz**  |
| on/ona  | **je**    |
| my      | **jemy**  |
| wy      | **jecie** |
| oni/one | **jedzą** |

### pić — 喝 `boire / trinken`

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **piję**    |
| ty      | **pijesz**  |
| on/ona  | **pije**    |
| my      | **pijemy**  |
| wy      | **pijecie** |
| oni/one | **piją**    |

### mieć — 有 `avoir / haben`

| 人称    | 变位      |
| :------ | :-------- |
| ja      | **mam**   |
| ty      | **masz**  |
| on/ona  | **ma**    |
| my      | **mamy**  |
| wy      | **macie** |
| oni/one | **mają**  |

### lubić — 喜欢 `aimer / mögen`

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **lubię**   |
| ty      | **lubisz**  |
| on/ona  | **lubi**    |
| my      | **lubimy**  |
| wy      | **lubicie** |
| oni/one | **lubią**   |

### mówić — 说 `parler / sprechen`

| 人称    | 变位        |
| :------ | :---------- |
| ja      | **mówię**   |
| ty      | **mówisz**  |
| on/ona  | **mowi**    |
| my      | **mówimy**  |
| wy      | **mówicie** |
| oni/one | **mówią**   |

用 mówić 表达"以某种语言说话"时,  使用 **po + 副词** 的结构: 

| 波兰语           | 德语            | 含义       |
| :--------------- | :-------------- | :--------- |
| **po polsku**    | auf Polnisch    | 说波兰语   |
| **po angielsku** | auf Englisch    | 说英语     |
| **po niemiecku** | auf Deutsch     | 说德语     |
| **po francusku** | auf Französisch | 说法语     |
| **po chińsku**   | auf Chinesisch  | 说中文     |
| **po rosyjsku**  | auf Russisch    | 说俄语     |
| **po włosku**    | auf Italienisch | 说意大利语 |

例句: 

|        |                           |
| :----- | :------------------------ |
| **PL** | Ona mówi po polsku.       |
| **DE** | Sie spricht auf Polnisch. |
| **FR** | Elle parle polonais.      |
| **中** | 她说波兰语.               |

> 注意观察变位规律: **ty (你)** 的变位通常以 **-sz** 结尾 (jesz, pijesz, masz, lubisz, mówisz) ,  **my (我们)** 以 **-my** 结尾,  **wy (你们)** 以 **-cie** 结尾. 这套规律适用于绝大多数波兰语动词,  相比德语的强/弱变化要规则得多. 
{: .prompt-info }

## 动物词汇 `Tier / animal`

波兰语中"动物"一词是 **zwierzę** (中性名词). 以下是常见的动物词汇: 

| 波兰语     | 德语        | 法语       | 含义 |
| :--------- | :---------- | :--------- | :--- |
| **pies**   | der Hund    | le chien   | 狗   |
| **kot**    | die Katze   | le chat    | 猫   |
| **koń**    | das Pferd   | le cheval  | 马   |
| **słoń**   | der Elefant | l'éléphant | 象   |
| **kaczka** | die Ente    | le canard  | 鸭   |
| **lew**    | der Löwe    | le lion    | 狮   |
| **mysz**   | die Maus    | la souris  | 鼠   |
| **tygrys** | der Tiger   | le tigre   | 虎   |
| **pająk**  | die Spinne  | l'araignée | 蜘蛛 |
| **ptak**   | der Vogel   | l'oiseau   | 鸟   |
| **ryba**   | der Fisch   | le poisson | 鱼   |

> 注意动物名词的性别: pies、kot、koń、słoń、lew、tygrys、pająk、ptak、ryba 以辅音结尾,  为阳性；kaczka、mysz 以 -a 结尾,  为阴性. 动物名词的性别不一定与实际生物性别对应. 
{: .prompt-tip }

## 形容词的性数一致

波兰语的形容词需要与所修饰的名词在性别和数上保持一致. 以 **smaczny** (美味的 / lecker / délicieux) 为例: 

| 搭配               | 性别 | 含义       |
| :----------------- | :--- | :--------- |
| **smaczny sok**    | 阳性 | 美味的果汁 |
| **smaczne wino**   | 中性 | 美味的酒   |
| **smaczna zupa**   | 阴性 | 美味的汤   |
| **smaczne jabłko** | 中性 | 美味的苹果 |

> 形容词词尾变化规律: 阳性为 **-y** (或 -i) ,  阴性为 **-a**,  中性为 **-e** (或 -ie). 这与名词的性别标记高度对应. 
{: .prompt-info }

## 名词复数: 词尾的变化

法语的复数通常只需加 -s,  德语的复数规则因词而异,  波兰语的复数变化则更为多样. 以下是几组常见例子: 

| 单数            | 复数            | 德语参照    | 规律提示        |
| :-------------- | :-------------- | :---------- | :-------------- |
| **kobieta**     | **kobiety**     | die Frauen  | 阴性 -a → -y    |
| **mężczyzna**   | **mężczyźni**   | die Männer  | 阳性,  内部变音 |
| **chłopiec**    | **chłopcy**     | die Jungs   | 阳性,  词尾变化 |
| **dziewczynka** | **dziewczynki** | die Mädchen | 阴性 -a → -i    |
| **dziecko**     | **dzieci**      | die Kinder  | 中性,  不规则   |
| **kot**         | **koty**        | die Katzen  | 阳性,  加 -y    |
| **pies**        | **psy**         | die Hunde   | 阳性,  不规则   |

## 句型示例

### 存在句 — "这是……"

用 **To jest** 表达"这是……": 

|        |                   |
| :----- | :---------------- |
| **PL** | To jest **dom**.  |
| **DE** | Das ist ein Haus. |
| **FR** | C'est une maison. |
| **中** | 这是一间房子.     |

### 定义句 — "……是……"

用 **to** 连接主语和定义时,  名词保持主格形式: 

|        |                          |
| :----- | :----------------------- |
| **PL** | Warszawa to **miasto**.  |
| **DE** | Warschau ist eine Stadt. |
| **FR** | Varsovie est une ville.  |
| **中** | 华沙是一座城市.          |

> 注意 **to** 的两种用法: **To jest dom** (这是一间房子) 和 **Warszawa to miasto** (华沙是一座城市). 前者 to 意为"这个",  后者 to 起系动词作用,  相当于"is". 这是波兰语中非常常见且实用的句式. 
{: .prompt-tip }

### 工具格与 "to" 连用

用 **To jest** 定义身份时,  若后面接工具格形式: 

|        |                                             |
| :----- | :------------------------------------------ |
| **PL** | To jest **tygrys**. → To jest **tygrysem**. |
| **DE** | Das ist ein Tiger.                          |
| **FR** | C'est un tigre.                             |
| **中** | 这是一只老虎.  (tygrys → tygrysem,  工具格) |

> 当使用 "to jest + 工具格" 时,  表示"这是 (一个) ……". 工具格词尾变化因性别而异: 阳性辅音结尾通常加 **-em** (tygrys → tygrysem) ,  阴性 -a 变为 **-ą** (kobieta → kobietą) ,  中性 -o 变为 **-em** (dziecko → dzieckiem). 这个规则将在后续课程中详细展开. 
{: .prompt-info }

### 动作句

|        |                      |
| :----- | :------------------- |
| **PL** | On je **jabłko**.    |
| **DE** | Er isst einen Apfel. |
| **FR** | Il mange une pomme.  |
| **中** | 他在吃苹果.          |

|        |                            |
| :----- | :------------------------- |
| **PL** | **Chłopiec** je **chleb**. |
| **DE** | Der Junge isst Brot.       |
| **FR** | Le garçon mange du pain.   |
| **中** | 男孩在吃面包.              |

## 综合练习: 三个小场景

现在把本课学到的知识放进生活场景里,  由浅入深.

### 场景一: 自我介绍

|        |                                                               |
| :----- | :------------------------------------------------------------ |
| **PL** | Cześć! Jestem **kobietą**. Mam **kota** i lubię **kawę**.     |
| **DE** | Hallo! Ich bin eine Frau. Ich habe eine Katze und mag Kaffee. |
| **FR** | Salut! Je suis une femme. J'ai un chat et j'aime le café.     |
| **中** | 你好! 我是一个女人. 我有一只猫,  我喜欢咖啡.                  |

> 练习要点: 问候 (Cześć) 、być + 工具格 (kobieta → kobietą) 、mieć 的第一人称变位 (mam) 、lubić 的第一人称变位 (lubię) 、动物词汇 (kot) 、饮品词汇 (kawa). 
{: .prompt-tip }

### 场景二: 餐桌

|        |                                                                             |
| :----- | :-------------------------------------------------------------------------- |
| **PL** | Na śniadanie jem **smaczny chleb** i piję **herbatę**. Ona pije **sok**.    |
| **DE** | Zum Frühstück esse ich leckeres Brot und trinke Tee. Sie trinkt Saft.       |
| **FR** | Au petit-déjeuner je mange du bon pain et je bois du thé. Elle boit du jus. |
| **中** | 早餐我吃美味的面包,  喝茶. 她喝果汁.                                        |

> 练习要点: 三餐 (śniadanie) 、食品词汇 (chleb) 、饮品词汇 (herbata,  sok) 、形容词一致 (smaczny + chleb 阳性) 、jeść / pić 的第一和第三人称变位 (jem / piję / pije) 、阴性宾格 (herbata → herbatę). 
{: .prompt-tip }

### 场景三: 在公园

|        |                                                                                               |
| :----- | :-------------------------------------------------------------------------------------------- |
| **PL** | To jest **pies**. **Dzieci** lubią **psy**. My mówimy po polsku i one mówią po polsku.        |
| **DE** | Das ist ein Hund. Kinder mögen Hunde. Wir sprechen auf Polnisch und sie sprechen Polnisch.    |
| **FR** | C'est un chien. Les enfants aiment les chiens. Nous parlons polonais et ils parlent polonais. |
| **中** | 这是一只狗. 孩子们喜欢狗. 我们说波兰语,  他们也说波兰语.                                      |

> 练习要点: 存在句 (To jest) 、动物词汇 (pies) 、不规则复数 (dziecko → dzieci,  pies → psy) 、lubić 的复数形式 (lubią) 、mówić + po polsku 语言表达、连词 i.
{: .prompt-tip }
