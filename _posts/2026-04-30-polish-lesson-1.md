## 认识波兰语的"性别世界"

波兰语的词性分为三种: 阳性 (męski) 、阴性 (żeński) 和中性 (nijaki).

先来看一组基础词汇.

| 波兰语          | 法语       |
| :-------------- | :--------- |
| **mężczyzna**   | un homme   |
| **chłopiec**    | un garçon  |
| **kobieta**     | une femme  |
| **dziewczynka** | une fille  |
| **dziecko**     | un enfant  |
| **człowiek**    | un humain  |
| **mleko**       | du lait    |
| **chleb**       | du pain    |
| **woda**        | de l'eau   |
| **ciasteczko**  | un biscuit |

> 以  **-a** 结尾的名词通常是阴性, 以 **-o** 或 **-e** 结尾的通常是中性, 以辅音结尾的则多为阳性. 不过例外也不少——mężczyzna (男人) 以 -a 结尾, 却是阳性名词.

## 格变化: 名词的形态变化

波兰语的名词在句子中承担不同语法角色时, 词尾会随之变化.

阴性名词 (-a 结尾) 作宾语时, 词尾 -a 变为 -ę.

|        |                       |
| :----- | :-------------------- |
| **PL** | On pije **wodę**.     |
| **FR** | Il boit **de l'eau**. |

而中性名词作宾语时, 形式保持不变:

|        |                         |
| :----- | :---------------------- |
| **PL** | On je **jabłko**.       |
| **FR** | Il mange **une pomme**. |

> 波兰语有 7 个格, 比拉丁语和俄语还要多. 不过第一课只需要认识**主格 (Mianownik)**、**宾格 (Biernik)** 和**工具格 (Narzędnik)** 即可.
{: .prompt-info }

## być

在大多数语言中, "be 动词"都是最常用、也最需要优先掌握的不规则动词. 波兰语中这个动词是 **być**.

六个人称的变位形式——三语对照如下:

| 人称    | 波兰语 być    | 法语 être |
| :------ | :------------ | :-------- |
| ja      | **jestem**    | suis      |
| ty      | **jesteś**    | es        |
| on,ona  | **jest**      | est       |
| my      | **jesteśmy**  | sommes    |
| wy      | **jesteście** | êtes      |
| oni,one | **są**        | sont      |

> 用 być 表达身份时, 后面的名词需要使用**工具格 (Narzędnik)**. 例如 Ja jestem kobietą (我是女人) ——kobietą 是 kobieta 的工具格形式. 这一点与德语和法语不同, 后两者在系动词后直接使用主格或表语名词.
{: .prompt-tip }

## 基础句型: "这是……" 与 "这不是……"

### To jest...

用 **To jest** 表达"这是……":

|        |                    |
| :----- | :----------------- |
| **PL** | To jest **chleb**. |
| **FR** | C'est **du pain**. |

### To nie jest...

用 **To nie jest** 表达"这不是……":

|        |                            |
| :----- | :------------------------- |
| **PL** | To nie jest **woda**.      |
| **FR** | Ce n'est **pas de l'eau**. |

> **To jest** 后面接主格名词, 表示"这是某物". 否定形式是 **To nie jest**. 这是最基础的存在句结构.
{: .prompt-tip }

## 日常问候与短语

以下是波兰语中最常用的日常用语, 掌握这些就可以应付最基本的交际场景.

### 问候与告别

| 波兰语          | 法语         |
| :-------------- | :----------- |
| **Cześć**       | Salut        |
| **Witaj**       | Bienvenue    |
| **Dzień dobry** | Bonjour      |
| **Dobranoc**    | Bonne nuit   |
| **Do widzenia** | Au revoir    |
| **Papa**        | Salut / Ciao |

### 礼节用语

| 波兰语          | 法语            |
| :-------------- | :-------------- |
| **Dziękuję**    | Merci           |
| **Proszę**      | Je vous en prie |
| **Przepraszam** | Pardon          |

### 基础回答

| 波兰语     | 法语 |
| :--------- | :--- |
| **Tak**    | Oui  |
| **Nie**    | Non  |
| **Dobrze** | Bien |
